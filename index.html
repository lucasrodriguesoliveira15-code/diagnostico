html e uma linguagem de programação

HTML não é considerado uma linguagem de programação, pois não possui lógica de controle, como loops ou condições. Ele é uma linguagem de marcação usada para estruturar o conteúdo da web, definindo como os elementos devem ser apresentados no navegador. Embora seja essencial para o desenvolvimento web, ele apenas organiza e descreve o conteúdo, sem executar operações lógicas.
